# CS50 Love2d Tour

This project is implementation of [CS50's Game Development Course](https://cs50.harvard.edu/games/2018/).

Each folder contains the game source files which is written in Lua(Love2d), with the assignment features implemented.

## Progression

- Chapter 0. Pong
  - [x] Make Pong
  - [x] (Optional) Make Ball's trailing effect
- Chapter 1. Flappy Bird
  - [x] More complicated procedure generation of pipe (random gap, random pipe generation time).
  - [x] Give the player a medal for their score.
  - [x] Pause Feature
- Chapter 2. Breakout
  - [x] Level Maker with flags(Alternate color, Skip Block, Mixed Level)
  - [ ] Block collision
  - [ ] Polishing game
